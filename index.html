Get all tokens JSON

Usage in HTML

<img src="https://yoursite.com/data/svg/btc.svg" alt="Bitcoin" width="32">

Usage in React / JSX

<img src={`${BASE_URL}/data/svg/${token.id}.svg`} alt={token.name} />

Fetch SVG code (for Figma plugins, etc.)

const res = await fetch('https://yoursite.com/data/svg/eth.svg'); const svgCode = await res.text(); // Paste into Figma, or inject into DOM
